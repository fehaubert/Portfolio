Frontend

Site de músicas

Esse site contém uma tela de log in não funcional, que o leva para a capa de uma playlist que você pode acessar e ouvir as músicas.

Site de músicas atualizado

Esse site contém uma tela de log in funcional, que o leva para o início de um "site de músicas", onde você tem acesso a uma playlist com algumas músicas.

Site de séries

Esse site contém páginas de Log In, cadastro, uma parte onde estão algumas séries e minhas notas para elas. Também conta com o "destaque da semana", que é Supernatural, e então uma página dedicada a série.

Página de Log In

Esse site contém uma única página que seria de Log In para a Cacau Show.

Indicações de séries

Esse site contém uma página mais simples com 5 indicações, com treilers, de séries na Netflix.

Backend

Calculadora

Esse site contém uma única tela, com uma calculadora simples funcional.

Casa de Hogwarts

Esse site contém uma única tela, com um espaço para inserir seu nome e um botão para sortear aleatoriamente uma das 4 casas de hogwarts para você, que aparecem como uma foto na página.

Tela de cadastro e Log In

Esse site contém páginas de cadastro, Log In e uma última tela com o nome que você inseriu no cadastro, tudo funcional.

Mobile

Responsividade

Esse site contém duas telas para um restaurante, sendo, a principio, para mobile, porém ela é responsiva.
